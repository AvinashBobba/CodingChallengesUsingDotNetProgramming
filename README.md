Dot Net Challenges and Interview coding exams

1. CaesarCipherService  : **The Caesar Cipher involves replacing each letter of the alphabet with the letter – standing places down or up according to the key given.**
   
   PROBLEM : Given "Zwddg ogjdv!" and a cipher key of 8, the result would be "Hello world!" and vice versa
   
   Inputs : input = "Zwddg ogjdv!" ,  number = 8

   Output : Hello World!
**--------------------------------------------------------------------------------------------------------------------------**
2. Morse Code  : **The Morse Code Convertor helps to convert a text to Morse code and Vice Vers **
   
   PROBLEM : Given "My Test String" with option A return (-- -.--/- . ... -/... - .-. .. -. --.) and with option B vice versa
   
   Inputs : input = My Test String

   Output : -- -.--/- . ... -/... - .-. .. -. --.
